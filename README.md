# User CRUD Application (React + Node.js + MongoDB)

A simple full-stack **User Management CRUD application** that allows you to Create, Read, Update, and Delete user records with the following fields:

- **Name**
- **Email**
- **Age**

This project uses **React (Vite)** for the frontend, **Node.js + Express** for the backend, and **MongoDB** for data storage.
